# Exercícios de Banco de Dados – MySQL

Este repositório contém todos os exercícios práticos de Banco de Dados desenvolvidos nas aulas.  
Cada atividade possui um diretório próprio dentro da pasta `exercicios_banco_de_dados/`, contendo:

- Arquivos `.sql` com criação de tabelas  
- Inserção de dados  
- Queries de seleção, filtro e relacionamentos  
- JOINs entre tabelas  
- Outros comandos trabalhados em aula


## Tecnologias Utilizadas
- **MySQL** (Workbench / CLI)
- **Modelagem relacional**
- **DDL (CREATE, ALTER, DROP)**
- **DML (INSERT, UPDATE, DELETE, SELECT)**
- **JOINs, filtros, LIKE, BETWEEN, operadores lógicos**

## Objetivo
Reforçar os conceitos de Banco de Dados Relacional através de prática com scripts SQL organizados por aula.

---

## Como Executar
Para executar qualquer exercício:

1. Abra o MySQL Workbench
2. Selecione o arquivo `.sql` da atividade desejada
3. Execute a criação do banco e tabelas
4. Execute os INSERTs
5. Rode as consultas da atividade
